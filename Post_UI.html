What is UI Design?

UI is the abbreviation of the phrase User Interface or translated into Vietnamese as user interface. It can be understood simply, all that is displayed on the screen when the user accesses the application is the UI such as colors, fonts, the layout of the website, the images of the web.

UI design is the person who designs the user interface to help users have the ability to communicate with computers as well as electronic devices through software, buttons, sliders, ... Or UI design is the decision maker. color, image of the website. In design, UI designs play the role of conveying messages from designers, providing services to customers by programming methods or building content so that users can view, understand, and use a particular content. effectively. Whether a website is beautiful or not, the layout is reasonable or not, whether it attracts readers or not depends on the design capabilities of the UI designs.
